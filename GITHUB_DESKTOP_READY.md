## 💻 Using VS Code + GitHub Desktop

### How They Work Together

**GitHub Desktop** handles:
- Creating branches
- Publishing repositories
- Pushing/pulling changes
- Viewing commit history

**VS Code** handles:
- Editing code
- Viewing files
- Running terminal commands
- Opening integrated git panel

### Daily Workflow

1. **Open GitHub Desktop**
   - See all your repositories

2. **Open in VS Code**
   - GitHub Desktop → Repository → Open in Visual Studio Code

3. **Edit code in VS Code**
   - Make changes, save files

4. **Back to GitHub Desktop**
   - See changed files
   - Write commit message
   - Click "Commit to master"

5. **Push to GitHub**
   - Click "Push Origin"
   - Code synced to GitHub ✅

### Alt: Do Everything in VS Code

1. **Open folder** in VS Code: `C:\Users\HARIHARAN\Projects\SentiPort`

   - See all tracked files
   - Stage/unstage changes
   - Write commit message
   - Click checkmark to commit

3. **Sync Changes**
   - VS Code automatically pushes/pulls from GitHub Desktop
