## 🔧 TROUBLESHOOTING

### "Python not found"
→ Install Python 3.8+ from python.org

### "Port in use"
→ Use different port or kill existing process

### "npm not found"
→ Install Node.js from nodejs.org

### "Flask import error"
→ Run: `pip install -r requirements.txt`
