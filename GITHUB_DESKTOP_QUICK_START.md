# ⚡ GitHub Desktop Quick Start (2 Minutes)

## 3 Simple Steps

### Step 1: Create GitHub Repository (1 minute)
```
1. Go to https://github.com/new
2. Name: SentiPort
3. Click "Create repository"
4. Copy the HTTPS URL shown
```

### Step 2: Open GitHub Desktop (30 seconds)
```
1. Launch GitHub Desktop
2. Click "Add" → "Add Local Repository"
3. Path: C:\Users\HARIHARAN\Projects\SentiPort
4. Click "Add Repository"
```

### Step 3: Push to GitHub (30 seconds)
```
1. In GitHub Desktop: "Publish repository"
2. Enter your GitHub username
3. Click "Publish Repository"
```

---

## ✅ Done!

Your code is now on GitHub! 🎉

## Now What?

**Option A: Edit in VS Code + Commit in GitHub Desktop**
```
1. Open in VS Code: GitHub Desktop → Repository → Open in Visual Studio Code
2. Make changes in VS Code
3. Back to GitHub Desktop → Write message → Commit
4. GitHub Desktop → Push Origin
```

**Option B: Do Everything in VS Code**
```
1. Open folder in VS Code: C:\Users\HARIHARAN\Projects\SentiPort
2. Make changes
3. Left sidebar → Source Control (Ctrl+Shift+G)
4. Stage files → Write message → Commit
5. Click "Sync Changes"
```

## Team Collaboration

**Share with team:**
```
1. On GitHub.com: Settings → Collaborators
2. Add team members by username
3. They can clone: git clone [YOUR_REPO_URL]
4. Everyone can push/pull
```
